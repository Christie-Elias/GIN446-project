Beyond HTML5

The day after the W3C unveiled the HTML5 logo, the WHATWG announced that HTML is now the new HTML5, deciding to stop using version numbers so they can fix bugs and add new features immediately rather than waiting for the next version.
The WHATWG and W3C are still working to preserve a "living standard," even if HTML5 is still the official standard. This indicates that HTML gets updated often rather than waiting for a major release like HTML6. As a result, the language is currently undergoing continuous, small-scale enhancements rather than being merely "newer than HTML5".

Arabic in URLs

2005: IRIs (RFC 3987) allow Arabic letters in URL paths (percent-encoded).
2010: ICANN approves Arabic domain names (IDNs) with Punycode.
2013: First Arabic TLDs launched (e.g., .مصر, .السعودية).
